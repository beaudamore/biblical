# Christian FOSS Literature Collection

This repository contains markdown versions of various Christian books, texts, and literature found in the Christian FOSS community. All content is derived from freely available and open source Christian projects.

## Contents

### Bible Texts
- `king-james-bible.md` - King James Version (Public Domain)
- `world-english-bible.md` - World English Bible (Public Domain)
- `american-standard-version.md` - American Standard Version 1901 (Public Domain)
- `hebrew-bible-overview.md` - Overview of the Open Scriptures Hebrew Bible

### Christian Confessions & Catechisms
- `westminster-confession.md` - Westminster Confession of Faith
- `westminster-shorter-catechism.md` - Westminster Shorter Catechism  
- `westminster-larger-catechism.md` - Westminster Larger Catechism
- `belgic-confession.md` - Belgic Confession
- `heidelberg-catechism.md` - Heidelberg Catechism
- `apostles-creed.md` - Apostles' Creed
- `nicene-creed.md` - Nicene Creed

### Hymns & Spiritual Songs
- `classic-christian-hymns.md` - Collection of classic Christian hymns
- `christmas-hymns.md` - Christmas hymns and carols
- `easter-hymns.md` - Easter and Lenten hymns

### Historical Christian Texts
- `95-theses.md` - Martin Luther's 95 Theses
- `second-helvetic-confession.md` - Second Helvetic Confession
- `canons-of-dort.md` - Canons of Dort

## Sources

All content is sourced from the following open source Christian projects:

- [Christian FOSS](https://github.com/meichthys/christian_foss) - Repository of Christian open source software
- [Open Scriptures Hebrew Bible](https://github.com/openscriptures/morphhb) - Open Scriptures Hebrew Bible Project
- [Open Bibles](https://github.com/seven1m/open-bibles) - Public domain Bible translations
- [Reformed Standards Compendium](https://github.com/reformed-standards/compendium) - Reformed confessions and catechisms
- [Open Hymnal Project](http://openhymnal.org/) - Freely distributable Christian hymns
